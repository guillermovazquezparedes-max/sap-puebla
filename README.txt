# Config para GitHub Pages (Vite + React)

1) Copia `vite.config.js` a la raíz de tu proyecto.
2) Copia `.github/workflows/deploy.yml` a la misma ruta en tu repo.
3) Haz commit y push a `main`.
4) En GitHub → Settings → Pages, selecciona **Source: GitHub Actions**.
5) Tu sitio quedará en `https://TU-USUARIO.github.io/sap-puebla/`.
